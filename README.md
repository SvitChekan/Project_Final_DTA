# Аналіз даних ecommerce платформ BigQuery та Kaggle

Цей проєкт присвячений перетворенню сирих даних таблиць `thelook_ecommerce` на зрозумілі бізнес-метрики та комплексному аналізу даних інтернет-магазину `ecommerce`. 

## Основні етапи роботи
* **SQL-запити:** Використання SQL-конструкцій
* **Візуалізація:** Побудова інтерактивного дашборду у Tableau Public.
* **Презентація:** Підготовка звіту з висновками та рекомендаціями.

## Технічний стек
* **SQL:** PostgreSQL
* **BI-інструменти:** Tableau
* **Документація:** PDF (PowerPoint)
