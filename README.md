# Аналіз даних ecommerce з BigQuery та Kaggle

Цей проєкт присвячений перетворенню "сирих" даних таблиць `thelook_ecommerce` на зрозумілі бізнес-метрики та комплексному аналізу даних інтернет-магазину `ecommerce`. 

### ℹ️ Hinweis zur Dokumentation
Das Projekt wurde im Rahmen meiner Weiterbildung ursprünglich auf Ukrainisch dokumentiert. Die technische Methodik, die Datenstruktur sowie der gesamte SQL-Code sind jedoch fachlich standardisiert und für eine internationale Auswertung optimiert.
